# 사용자 매뉴얼

## 1. 누가 무엇을 해야 하나

### 일반 사용자

일반 사용자는 Instagram 계정을 연결하고 게시 작업을 만듭니다.

일반 사용자가 알 필요가 없는 것:

- 관리자 설정 키
- Meta App ID
- Meta App Secret
- Cloudflare Secret
- D1, KV, R2 같은 인프라 설정

### 운영자

운영자는 앱이 정상 작동하도록 Meta와 Cloudflare 설정을 관리합니다.

운영자가 관리하는 것:

- Meta Developer App
- Facebook Login for Business
- OAuth Redirect URI
- Cloudflare Secret
- D1/KV 바인딩

## 2. 새 사용자가 Instagram 계정을 연결하는 방법

1. `https://automatic-posting.pages.dev`에 접속합니다.
2. 오른쪽 `계정 연결` 영역을 확인합니다.
3. `Instagram 연결하기`를 누릅니다.
4. Facebook 로그인 화면이 열리면 본인 Facebook 계정으로 로그인합니다.
5. 게시할 Facebook Page와 Instagram Business 계정을 선택합니다.
6. 권한 승인 화면에서 게시 권한을 허용합니다.
7. 앱으로 돌아오면 연결된 Instagram 계정명이 표시됩니다.
8. 게시 채널에 Instagram이 표시되고 선택되면 준비 완료입니다.

연결이 되지 않을 때 확인할 것:

- Instagram 계정이 프로페셔널 계정인지 확인합니다.
- Instagram 계정이 Facebook Page에 연결되어 있는지 확인합니다.
- 현재 Facebook 계정이 해당 Page에 충분한 권한을 갖고 있는지 확인합니다.
- 권한 승인 화면에서 Page와 Instagram 계정을 빠뜨리지 않았는지 확인합니다.

## 3. 바로 게시하는 방법

1. `캠페인 기본값`에서 게시 채널을 확인합니다.
2. 제목과 본문을 입력합니다.
3. 해시태그와 링크가 있으면 입력합니다.
4. 이미지를 선택하거나 비워둡니다.
5. 이미지를 비워두면 앱이 글 내용을 기반으로 Instagram용 JPG 이미지를 자동 생성합니다.
6. 발행 모드를 `즉시 발행`으로 둡니다.
7. `게시 작업 만들기`를 누릅니다.
8. `발행 작업` 영역에서 성공/실패 상태를 확인합니다.

## 4. 예약 게시하는 방법

1. 제목, 본문, 해시태그, 링크를 기본값으로 입력합니다.
2. `날짜 폴더 예약`에서 상위 폴더를 선택합니다.
3. 폴더명은 날짜 형식이어야 합니다.

```text
2026-06-24
2026-06-25
```

4. 각 날짜 폴더 안에 이미지를 넣습니다.
5. 파일명 숫자순으로 예약 순서가 정해집니다.
6. 시작 시간과 간격을 설정합니다.
7. 예약 계획을 확인합니다.
8. `예약 작업 만들기`를 누릅니다.
9. `발행 작업` 영역에서 예약 상태를 확인합니다.

## 5. 캡션 파일 사용법

이미지별 문구를 다르게 쓰려면 이미지와 같은 이름의 텍스트 파일을 넣습니다.

```text
001.jpg
001.txt
002.jpg
002.txt
```

텍스트 파일 예시:

```text
title: 첫 번째 게시글
body: 오늘 소개할 내용입니다.
hashtags: #브랜드 #소식
link: https://example.com
```

CSV를 사용할 수도 있습니다.

```text
captions.csv
```

권장 컬럼:

```text
file,title,body,hashtags,link
```

## 6. 시간 기준

예약 시간은 브라우저 현지 시간 기준입니다. 한국에서 사용하면 보통 KST 기준으로 예약됩니다.

## 7. 운영자에게 요청해야 하는 경우

다음 상황은 일반 사용자가 직접 해결하지 않고 운영자에게 요청합니다.

- Instagram 연결 버튼이 보이지 않음
- Facebook 로그인 자체가 사용할 수 없다고 나옴
- 연결 가능한 Page 또는 Instagram 계정이 보이지 않음
- 앱 전체가 운영 설정 확인 필요 상태로 표시됨
- Meta 권한 검수 또는 앱 설정 변경이 필요함

운영자에게 전달할 정보:

- 사용한 Facebook 계정
- 연결하려는 Instagram username
- 연결하려는 Facebook Page 이름
- 화면에 표시된 오류 문구
- 문제가 발생한 시간
