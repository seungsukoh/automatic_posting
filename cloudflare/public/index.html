게시 계정 연결

Instagram과 Threads 계정을 연결해 이미지와 글을 게시합니다.
연결 상태를 확인하고 있습니다.
연결 정보 입력 필요한 앱 연결값만 입력합니다.
설정 상태를 확인하고 있습니다.
입력값 확인 가이드
Threads만 연결할 때 아래 Callback URL을 Meta 앱의 Valid OAuth Redirect URIs에 그대로 추가합니다. Threads App ID와 Threads App Secret은 Meta for Developers의 해당 Threads 앱, 앱 설정, 기본 화면에서 확인합니다.
Instagram도 연결할 때 Meta App ID와 Meta App Secret은 Instagram Business 연결에 쓰는 Meta 앱의 앱 설정, 기본 화면 값입니다. Facebook Login Configuration ID는 Facebook Login for Business 설정 화면의 Configuration ID입니다.
입력하지 않는 값 Threads 표시 이름은 Meta 콘솔 표시용이라 이 앱에 저장하지 않습니다. 제거 콜백 URL과 삭제 콜백 URL은 이 연결 흐름에 필요하지 않습니다. Meta가 필수로 요구할 때만 별도 콜백을 준비합니다.
연결 전 확인 Callback URL은 끝에 슬래시 없이 정확히 등록해야 합니다. 앱이 테스트 모드이면 연결할 Threads 계정이 앱 역할 또는 테스터 초대를 수락해야 합니다.
