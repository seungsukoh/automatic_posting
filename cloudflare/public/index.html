Connection setup

관리자 연결 설정

이 설정은 운영자가 한 번만 저장합니다. 일반 사용자는 설정값을 입력하지 않고 계정 연결 버튼만 사용합니다.

1 필요한 값

관리자 설정 키, AutoPosting의 Meta 앱 ID 1505779147711821, 같은 앱의 App Secret을 준비합니다.

2 Redirect URI

https://automatic-posting.pages.dev/api/auth/meta/callback

3 입력 위치

아래 폼에 값을 저장한 뒤 계정 연결에서 Instagram 또는 Threads를 승인합니다.

Meta App ID

Meta for Developers > My Apps > AutoPosting > App settings > Basic > App ID

아래 Meta App ID 칸에는 1505779147711821을 넣습니다.
Meta App Secret

Meta for Developers > My Apps > AutoPosting > App settings > Basic > App secret

아래 Meta App Secret 칸에는 AutoPosting 앱의 App Secret을 넣습니다.
Redirect URI

https://automatic-posting.pages.dev/api/auth/meta/callback

Facebook Login for Business 설정의 OAuth Redirect URIs에 등록
관리자 설정 키

Cloudflare Pages > Settings > Environment variables > ADMIN_SETUP_KEY

아래 관리자 설정 키 입력칸에 붙여넣기
  1. Meta for Developers에 로그인하고 앱을 선택합니다.

    이미 만든 앱이 있으면 My Apps에서 해당 앱을 엽니다. 새로 만들 경우 비즈니스 연결 목적의 앱을 생성하고 연결할 Facebook Page, Instagram Business 계정, Threads 계정이 같은 비즈니스 자산에 있는지 확인합니다.

  2. Meta App ID를 확인합니다.

    App settings > Basic 화면에서 AutoPosting 앱 ID 1505779147711821을 복사합니다. 이 앱 ID로 Facebook Login for Business를 시작합니다.

  3. Meta App Secret을 확인합니다.

    같은 Basic 화면에서 App Secret 값을 표시하고 복사합니다.

  4. Instagram과 Facebook Login for Business를 설정합니다.

    앱 대시보드에서 Instagram API와 Facebook Login for Business를 설정합니다. Instagram 게시에는 Business 계정과 instagram_basic, instagram_content_publish, pages_show_list, pages_read_engagement, business_management 권한이 필요합니다.

  5. OAuth Redirect URI를 등록합니다.

    Facebook Login for Business 설정의 OAuth Redirect URIs에 위 Redirect URI를 그대로 추가합니다. 도메인, 경로, 끝의 /callback까지 일치해야 합니다.

  6. 관리자 설정 키를 확인합니다.

    Cloudflare Pages 프로젝트의 Environment variables 또는 Secrets에서 ADMIN_SETUP_KEY 값을 확인합니다. 이 값은 Meta app 값을 앱에 저장할 권한 확인용이며 Meta 콘솔 값이 아닙니다.

  7. 아래 폼에 세 값을 저장합니다.

    관리자 설정 키, Meta App ID 1505779147711821, AutoPosting App Secret을 입력하고 설정 저장을 누릅니다. 저장 후 상태 배지에서 Facebook Login이 설정됨으로 바뀌면 계정 승인 준비가 된 상태입니다.

  8. 계정 연결 영역에서 Instagram 또는 Threads를 승인합니다.

    연결 설정 창을 닫고 계정 연결 카드의 연결하기를 누릅니다. Meta 승인 화면에서 게시할 계정과 필요한 권한을 선택하면 앱으로 돌아와 연결됨 상태가 표시됩니다.

설정 상태를 확인하고 있습니다.
