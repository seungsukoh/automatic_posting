Instagram / Threads publishing

Automatic Posting

이미지가 있으면 Instagram에, 글만 있으면 Threads에 게시하고 Instagram 날짜 폴더 예약도 만듭니다.

계정 확인 글 작성 결과 확인
1계정 연결 2글 작성 3예약 4결과

Step 1

게시 계정 연결

Instagram은 이미지와 글 게시, Threads는 글 전용 게시용으로 연결합니다.
연결 상태를 확인하고 있습니다.
연결 설정 Instagram / Threads 연결값을 입력한 뒤 계정을 연결합니다.
설정 상태를 확인하고 있습니다.
A
연결 정보 연결에 필요한 Redirect URI와 설정 상태를 확인합니다.
OAuth Redirect URI
M
Meta / Instagram Instagram Business 연결에 사용하는 Meta 앱 설정입니다.
T
Threads Threads 계정 연결과 글 게시에 사용하는 앱 설정입니다.

Step 2

글 작성

이미지를 선택하지 않으면 글 내용을 기반으로 Instagram용 JPG 이미지를 자동 생성합니다.
대기 중
1
게시 채널 연결된 계정만 선택할 수 있습니다.
플랫폼
2
기본 문구 캡션 파일이 없는 이미지에 적용됩니다.
3
캠페인 추적 필요할 때만 입력합니다.
태그, 목표, UTM 설정
바로 게시 글만 입력해도 Instagram용 이미지를 자동 생성합니다.
이미지
선택된 이미지가 없습니다.
게시 미리보기 플랫폼별 최종 문구

Step 3

날짜 폴더 예약

날짜 폴더의 이미지를 번호순으로 예약합니다.
대기 중
폴더 날짜 폴더명 기준
순서 파일명 숫자순
시간 시작 시간 + 간격
선택된 폴더가 없습니다.

Step 4

발행 작업

최근 작업 상태와 실패 사유를 확인합니다.
